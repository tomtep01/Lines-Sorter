# Lines Sorter

Lines Sorter is a program built to read every line, see if the line has matching contents, then save it.

## Building from Source

To build this program, you need:

*   **Prerequisites:**
    *   .NET Framework 4.8 is recommended
    *   Microsoft Visual Studio

*   **Steps:**
    1.  Clone the repo.
    ```git clone https://github.com/tomtep01/Lines-Sorter```
    2.  Open the solution file in Visual Studio.
    3.  Build the solution.
